## Usage Example

```go
package main

import (
    "context"
    "github.com/kawai-network/veridium/pkg/localfs"
)

func main() {
    service := localfs.NewService()
    ctx := context.Background()
    
    // Write a file
    service.WriteFile(ctx, localfs.WriteFileParams{
        Path:    "/tmp/test.txt",
        Content: "Hello, World!",
    })
    
    // Read the file
    result, _ := service.ReadFile(ctx, localfs.ReadFileParams{
        Path: "/tmp/test.txt",
    })
    
    println(result.Content)
}
```

## Key Design Decisions

1. **Context-Aware**: All operations accept `context.Context` for cancellation
2. **Error Handling**: Explicit error returns following Go conventions
3. **Type Safety**: Strongly typed parameters and results
4. **Platform Abstraction**: Platform-specific code isolated in separate files
5. **JSON Compatible**: All types have JSON tags for serialization
6. **Service Pattern**: Stateful service with method receivers
7. **Background Commands**: Shell commands stored in service for management
